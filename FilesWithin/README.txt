Steps to run the project

1. Installation of Python 3,x
2. Installation of DJango
3. Installation of Crispy-Forms
4. Installation of PyCharm Community Version
5. Open the project in PyCharm IDE
6. Navigate to the project folder in terminal or command prompt
7. Run the localhost server using the command - python manage.py runserver
8. Open any web browser and navigate to the link - 127.0.0.1:8000 or localhost:8000
